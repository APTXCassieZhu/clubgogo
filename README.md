# clubgogo

# SpringBoot Maven + leaflet map
# a website for finding clubs in SBU
# 1. Can check the club activity in the map, by clicking building
# 2. Can search club by keyword, or by category
# 3. Can add favorite club to personal page, also can unfollow club in personal page or club page
# 4. Can set up personal information in the personal page
# 5. Can send email to club manager
